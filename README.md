# AI Navigation (AI 导航站)

这是一个具有商业潜力的 AI 工具导航站项目，经过 AI 技术加持（自然语言搜索）。

## 功能特点

- **精选 AI 工具**: 收录 ChatGPT, Midjourney 等热门工具。
- **AI 智能搜索**: 支持自然语言提问，例如 "我想画一张图" 或 "帮我写代码"，系统会自动推荐相关工具。
- **现代化 UI**: 使用 React + Tailwind CSS 构建，响应式设计，美观大方。

## 项目结构

- `client/`: 前端项目 (Vite + React + Tailwind CSS)
- `server/`: 后端项目 (Node.js + Express)

## 如何运行

### 1. 启动后端

```bash
cd server
npm install
node index.js
```

后端将运行在 http://localhost:3001

### 2. 启动前端

```bash
cd client
npm install
npm run dev
```

前端将运行在 http://localhost:5173

## 商业价值分析

此类导航站是目前 AI 浪潮中的流量金矿：
1.  **SEO 流量**: 聚合长尾关键词。
2.  **付费收录**: 允许新 AI 产品付费置顶或提交。
3.  **会员订阅**: 提供深度评测或独家优惠。
